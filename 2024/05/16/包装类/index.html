1: 为什么需要包装类

有些 jdk 方法需要的参数是 Object 类型，不让传入基本数据类型，比如：

equals(Object obj); 还有：集合的 add(Object)

包装类的父类是 Number;

2：基本数据类型和包装类在堆内存和栈内存的位置

基本数据类型存在栈中；

包装类：它有一个变量被声明在栈中，然后指向在堆中的分配的真正的值

![image-20240509171713235](C:\Users\Administrator\Nutstore\1\我的坚果云\笔记\面试准备\Java 基础\image-20240509171713235.png)

3：包装类和基本数据类型比较大小时

包装类会自动拆箱，因为成本低。

4：包装类和基本数据类型的转换

  1. 数字转包装类：装箱

    调用包装类的 api 方法 valueOf(1);

  2. 包装类转数字：拆箱

    包装类变量的 intValue();

  3. 字符串转为包装类

    调用包装类的 api 方法 parseInt(“1”);

5：包装类和基本数据类型谁占用的内存大？

包装类占用的内存大。包装类在内存中，除了数据本身外，还有对象头、对齐填充等信息
